Assignment 8 - Knowledge Representation

Knowledge Domain - Compiling a piece of C code

This consists of checking the syntactical validity of the C code as per standard guidelines

Knowledge Representation

The best form of representation for the above scenario is in the form of Production Rules.

To generate a string in the language, one begins with a string consisting of only a single start symbol, and then successively applies the rules (any number of times, in any order) to rewrite this string. This stops when we obtain a string containing only terminals. The language consists of all the strings that can be generated in this manner. Any particular sequence of legal choices taken during this rewriting process yields one particular string in the language. If there are multiple different ways of generating this single string, then the grammar is said to be ambiguous.

The advantages of Production Rules for my knowledge domain are:

  1. Very easy to understand and apply.
  2. Its implementation is straightforward in computer therefore automating the process.
  3. A formal definition in form of BNF includes metalanguage for the definition of language syntax.
  4. Most importantly, it focuses only on the syntactic description of the language.
Logic

Apart from the above form of Knowledge Representation, other forms such as Logic can also be applied.

It mainly has two parts to knowledge representation language:

  • Syntax – It describes the possible configurations that can constitute sentences.
  • Semantics – It determines the facts in the world to which the sentences refer and also tells us what the agent believes.

For my Knowledge Domain, the first part i.e. Syntax is more important. So, in the compiler that will compile the C – code, the syntax for various entities of C – language should be defined. For example, a variable name should not start with a number or special character. Similarly, every line of C terminates with a semicolon.

So, as per my understanding, Production Rule will help to focus at the ground level of defining the syntactical rules for compiling C – code which includes grammar, metalanguage etc. Whereas, Logic which contains two parts i.e. Syntax and Semantics will help me to look at the compiling issue from a higher level.
