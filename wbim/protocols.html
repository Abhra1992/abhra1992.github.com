Assignment 4 - Internet Protocols and Standards

Following are a list of some protocols and standards for the Internet and the web with their specification documents where applicable.

  • Secure Shell (SSH)
  • File Transfer Protocol (FTP)
  • Simple Mail Transfer Protocol (SMTP)
  • Telephone Network (Telnet)
  • HyperText Transfer Protocol (HTTP)
  • Secure HyperText Transfer Protocol (HTTPS)
  • Secure File Transfer Protocol (SFTP)
  • Secure Sockets Layer (SSL)
  • Transport Layer Security (TLS)
  • Post Office Protocol (POP)
  • Domain Name System (DNS)
  • Network Time Protocol (NTP)
  • Internet Relay Chat (IRC)
  • Lightweight Directory Access Protocol (LDAP)

An exhaustive list of protocols and their related specifications, ports and versions can be found in Wikipedia.
