Assignment 5 - Categorisation of WBIS

There are different ways of categorizing Web Based Information Systems :

  • Based on Function
  • Based on Organizational / Operational Role
  • Based on System Properties

However, the above categorisation system can be improved by specialising into 3 distinct levels :

Level 1
  • Small Scale : It refers to an organization which might have a single branch in a particular country.
  • Medium Scale : Identical to above.
  • Large Scale : It refers to a MNC which has multiple branches in multiple countries.
Level 2
  • Manual : Booking a ticket in the bus by paying the fee to the conductor and getting the ticket in return is an example.
  • Automatic : Online flight booking or the working of an ATM are very good examples of an automated system.
  • Manual + Automatic : Cash deposition in a bank which involves some manual paperwork followed by automated data entry via computer.
Level 3
  • Complexity : a less complex system is easy to play with and therefore more beneficial for the user.
  • Adaptability : A system that can incorporate the user feedback easily and therefore further enhance the system to a better level.
  • Interactive : The more interactive a system is, the more easily the user can communicate with the system and exchange information.
